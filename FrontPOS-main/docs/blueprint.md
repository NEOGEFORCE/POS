# **App Name**: ORG-POS

## Core Features:

- User Authentication: User authentication with 'administrador@gmail.com' (password '123456') for admin and 'empleado@gmail.com' (password '123456') for employee roles.
- User Management: Management of users and employees, including adding, editing, and deactivating user accounts.
- Customer Management: Management of customer information, including contact details and purchase history.
- Category Management: Management of product categories for organizing products.
- Product Management: Management of products, including descriptions, pricing, and inventory levels.
- Expense Tracking: Record and categorize expenses to track business spending.
- AI Product Recommendations: AI tool provides product recommendations based on customer purchase history and trends.
- Dashboard: Displays an overview of sales data and key metrics.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire trust.
- Background color: Very light blue (#E8EAF6) to maintain a professional, clean look with sufficient contrast.
- Accent color: A slightly brighter blue (#303F9F) for call-to-action buttons, and key UI elements.
- Font pairing: 'Poppins' (sans-serif) for headers, and 'PT Sans' (sans-serif) for body text.
- A set of clear and simple icons to represent actions and entities within the POS system.
- Clean, intuitive, and responsive layout that adapts to different screen sizes. Focus on usability, legibility and easy access to all core features.
- Subtle transitions and animations for UI elements and interactions to provide a smooth and engaging user experience. (e.g. loading spinners, button presses)
